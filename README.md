# NAFLD Biomarker Discovery Pipeline (Single-Dataset Path)

This repository contains an end-to-end bioinformatics pipeline for identifying stage-specific transcriptomic biomarkers in Non-Alcoholic Fatty Liver Disease (NAFLD). The project focuses on a 4-class staging architecture using the GSE135251 RNA-seq dataset.

## Project Overview

NAFLD progression is traditionally staged histologically. This pipeline aims to distinguish between four clinical stages using hepatic gene expression data:
- **Class 0: Healthy Control**
- **Class 1: Simple Steatosis (NAFL)**
- **Class 2: NASH (Non-Alcoholic Steatohepatitis)**
- **Class 3: Advanced Fibrosis (F2–F4)**

The pipeline is designed for high reproducibility, with each stage contained in a sequential Jupyter notebook.

## Key Features

1.  **GSE135251 RNA-seq Analysis**: Focused on a high-quality, large-scale (n=206 liver biopsies) dataset to minimize batch effects.
2.  **4-Class Staging**: Moving beyond binary classification to a clinically relevant 4-stage progression model.
3.  **Consensus Feature Selection**: Integrating multiple algorithms (Boruta, LASSO, Random Forest, XGBoost) to identify robust biomarkers.
4.  **Explainable AI**: Utilizing SHAP (SHalpey Additive exPlanations) for per-class biomarker attribution.
5.  **Biological Validation**: GO and KEGG pathway enrichment analysis to validate the biological relevance of identified biomarkers.

## Setup and Installation

### Prerequisites
- Python 3.10+
- Jupyter Notebook or JupyterLab

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Karim-Zeyada/nafld-single-dataset.git
    cd NAFLD-Biomarker-Discovery
    ```

2.  **Create a virtual environment**:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```

3.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

## Usage
Run the notebooks in sequential order (`NB01` to `NB07`). Each notebook is designed to be self-contained but depends on the outputs generated by previous steps.

## Data Source
- **GEO Accession**: [GSE135251](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE135251)
- **Platform**: Illumina HiSeq 2500 (Human liver biopsy)
- **Reference**: Govaere et al., *Science Translational Medicine*, 2020. [DOI: 10.1126/scitranslmed.aba4448](https://doi.org/10.1126/scitranslmed.aba4448)
